Comandos Útiles para Testing HATEOAS

🔍 Verificar Respuestas HATEOAS:

curl -H "Accept: application/hal+json" "http://localhost:9002/api/courses"
curl -H "Accept: application/hal+json" "http://localhost:9002/api/courses/1"
curl -H "Accept: application/hal+json" "http://localhost:9001/api/users"
curl -H "Accept: application/hal+json" "http://localhost:9003/api/student-marks"
curl -H "Accept: application/hal+json" "http://localhost:9004/api/payments"
curl -H "Accept: application/hal+json" "http://localhost:9005/api/support-tickets"

📋 Verificar OpenAPI Docs:

curl "http://localhost:9001/v3/api-docs"
curl "http://localhost:9002/v3/api-docs"
curl "http://localhost:9003/v3/api-docs"
curl "http://localhost:9004/v3/api-docs"
curl "http://localhost:9005/v3/api-docs"

🔌 Verificar Servicios Activos:

netstat -an | findstr :900
netstat -an | findstr :8761
