About EduTech Classroom Parent Project

Parent POM for all EduTech microservices and shared modules

Project Modules

This project has declared the following modules:

Name Description
Classroom - Eureka Module Netflix Eureka: Dynamic microservice discovery, registration, and lookup.
EduTech Common Library Librería compartida con DTOs, manejadores de excepción, configuración y utilidades para microservicios EduTech.
Classroom - Users Module User Service: user account management, authentication, and profiles.
Classroom - Courses Module Course Service: course information management and student enrollments.
Classroom - Grades Module Grades Service: manages the registration and lookup of student grades.
Classroom - Payments Module Payments Service: responsible for processing transactions, tuition payments, and billing.
Classroom - Support Module Support Service: support ticket management and handling of user inquiries.
